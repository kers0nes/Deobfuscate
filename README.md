# Lua Deobfuscator Studio

A web-based tool that implements Lua deobfuscation with loadstring simulation, code editor, and obfuscation dashboard.

## Features
- 🔓 Loadstring simulation & extraction
- 📝 Monaco-based code editor
- 🛡️ Obfuscation dashboard (educational)
- 🎮 Roblox script support
- ⚡ Real-time deobfuscation

## Usage
1. Paste obfuscated Lua/Roblox code
2. Click "Deobfuscate (Loadstring Sim)"
3. View cleaned code in dashboard
